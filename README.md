# Project-Sudoku-Puzzle-Generator
Brendan Chan
This is a program to generate a completed Sudoku board.
Run the program by copying the code in SudokuGenerator.java and pasting it into an IDE.
The Sudoku board is generated using a base 1d array and a simple math equation to move through the 1d array.
In this project there is the DesignDocument.docx, README.md and SudokuGenerator.java.
